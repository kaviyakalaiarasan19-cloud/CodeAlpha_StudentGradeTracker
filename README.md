# CodeAlpha Student Grade Tracker

**Menu-driven Java application** to manage and analyze student grades. This project allows users to add, view, update, delete, search, and sort student records efficiently. It also provides statistics and supports login, file saving, and PDF report export.

---

## Features

- **Add/View Students** – Add new student records and view all existing students.  
- **Search Students** – Search students by name.  
- **Update/Delete Students** – Update marks or delete student records.  
- **Grade Classification** – Automatically assign grades (A/B/C/Fail) based on marks.  
- **Statistics** – Calculate average, highest, and lowest marks.  
- **Sort Students** – Sort students by marks in ascending or descending order.  
- **File Saving/Loading** – Save student data to a file and load it on startup.  
- **Login System** – Simple username and password authentication.  
- **Export Report as PDF** – Generate a PDF report of all students with grades and statistics.
